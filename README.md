## How to use
To run Quesion Generation, you can simply use the command below:
```
./ask {PATH_TO_ARTICLE} {N_QUESTIONS}
```

To run Question Answer, use the command below:
```
./answer {PATH_TO_ARTICLE} {PATH_TO_QUESTIONS}
```
